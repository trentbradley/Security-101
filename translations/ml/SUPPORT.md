# പിന്തുണ

## പ്രശ്നങ്ങൾ റിപ്പോർട്ട് ചെയ്യാനും സഹായം നേടാനും എങ്ങനെ  

ഈ പ്രോജക്റ്റ് ബഗുകളും ഫീച്ചർ അഭ്യർത്ഥനകളും ട്രാക്ക് ചെയ്യാൻ GitHub Issues ഉപയോഗിക്കുന്നു. ഡുപ്ലിക്കേറ്റുകൾ ഒഴിവാക്കാൻ പുതിയ പ്രശ്നങ്ങൾ റിപ്പോർട്ട് ചെയ്യുന്നതിന് മുമ്പ് നിലവിലുള്ള പ്രശ്നങ്ങൾ തിരയുക. പുതിയ പ്രശ്നങ്ങൾക്കായി, നിങ്ങളുടെ ബഗ് അല്ലെങ്കിൽ ഫീച്ചർ അഭ്യർത്ഥന ഒരു പുതിയ Issue ആയി ഫയൽ ചെയ്യുക.

ഈ പ്രോജക്റ്റ് ഉപയോഗിക്കുന്നതിനെക്കുറിച്ചുള്ള സഹായത്തിനും ചോദ്യങ്ങൾക്കുമായി, **റിപ്പോ പരിപാലകൻ: സഹായത്തിനായി റിപ്പോ ഉടമകളുമായി അല്ലെങ്കിൽ കമ്മ്യൂണിറ്റിയുമായി എങ്ങനെ ബന്ധപ്പെടാമെന്ന് ഇവിടെ നിർദ്ദേശങ്ങൾ ചേർക്കുക. ഇത് ഒരു STACK OVERFLOW ടാഗ് അല്ലെങ്കിൽ മറ്റേതെങ്കിലും ചാനൽ ആയിരിക്കാം. നിങ്ങൾ എവിടെ ആളുകളെ സഹായിക്കും?**.

## Microsoft പിന്തുണ നയം  

ഈ **പ്രോജക്റ്റിന് അല്ലെങ്കിൽ ഉൽപ്പന്നത്തിന്** പിന്തുണ മുകളിൽ നൽകിയിരിക്കുന്ന വിഭവങ്ങളിൽ മാത്രം പരിമിതമാണ്.
