# ഇൻഫ്രാസ്ട്രക്ചർ സുരക്ഷാ ശേഷികൾ

ഈ വിഭാഗത്തിൽ, ഇൻഫ്രാസ്ട്രക്ചർ സുരക്ഷയിൽ ഉപയോഗിക്കുന്ന പ്രധാന ഉപകരണങ്ങളും ശേഷികളും സംബന്ധിച്ച കൂടുതൽ വിശദാംശങ്ങൾ പരിഗണിക്കും.

**ആമുഖം**

ഈ പാഠത്തിൽ, നാം പരിഗണിക്കുന്നത്:

- വൾണറബിലിറ്റി മാനേജ്മെന്റ് എന്താണ്?

- CNAPP എന്താണ്?

## വൾണറബിലിറ്റി മാനേജ്മെന്റ് എന്താണ്?

വൾണറബിലിറ്റി മാനേജ്മെന്റ് എന്നത് കമ്പ്യൂട്ടർ സിസ്റ്റങ്ങൾ, സോഫ്റ്റ്വെയർ, നെറ്റ്വർക്കുകൾ, ഇൻഫ്രാസ്ട്രക്ചർ എന്നിവയിൽ ഉള്ള ദുർബലതകളെ തിരിച്ചറിയുക, വിലയിരുത്തുക, മുൻഗണന നൽകുക, പരിഹരിക്കുക, നിരീക്ഷിക്കുക എന്നിവയുടെ ഒരു ക്രമബദ്ധമായ പ്രക്രിയയാണ്. വൾണറബിലിറ്റി മാനേജ്മെന്റിന്റെ പ്രധാന ലക്ഷ്യം, ദുർവിനിയോഗം ചെയ്യപ്പെടുന്നതിന് മുമ്പ് അറിയാവുന്ന ദുർബലതകളെ പ്രാക്ടീവായി പരിഹരിച്ച് ഒരു സ്ഥാപനത്തിന്റെ സുരക്ഷാ അപകടസാധ്യത കുറയ്ക്കുക എന്നതാണ്. ഇതിൽ ഉൾപ്പെടുന്ന പ്രധാന ഘട്ടങ്ങൾ:

തിരിച്ചറിയൽ: സെർവറുകൾ, ആപ്ലിക്കേഷനുകൾ, നെറ്റ്വർക്കുകൾ, എൻഡ്പോയിന്റുകൾ എന്നിവ ഉൾപ്പെടുന്ന സ്ഥാപനത്തിന്റെ ആസ്തികളിൽ ദുർബലതകളെ സ്കാൻ ചെയ്യുകയും കണ്ടെത്തുകയും ചെയ്യുക.

മൂല്യനിർണ്ണയം: Common Vulnerability Scoring System (CVSS) സ്കോർ, ബിസിനസ് പ്രാധാന്യം തുടങ്ങിയ ഘടകങ്ങൾ അടിസ്ഥാനമാക്കി ഓരോ ദുർബലതയുടെ ഗുരുത്വവും സാധ്യതയുള്ള സ്വാധിനവും വിലയിരുത്തുക.

മുൻഗണന: ദുർബലതകളെ അവരുടെ അപകടനിരക്ക്, സ്ഥാപനത്തിന്റെ പ്രവർത്തനങ്ങളിലേക്കും ഡാറ്റയിലേക്കും ഉള്ള സ്വാധിനം എന്നിവയുടെ അടിസ്ഥാനത്തിൽ റാങ്ക് ചെയ്യുക.

പരിഹാരം: പാച്ചുകൾ പ്രയോഗിക്കുക, സുരക്ഷാ ക്രമീകരണങ്ങൾ കോൺഫിഗർ ചെയ്യുക, അല്ലെങ്കിൽ പ്രതിരോധ നിയന്ത്രണങ്ങൾ വിന്യസിക്കുക എന്നിവ ഉൾപ്പെടുന്ന ദുർബലതകൾ പരിഹരിക്കാനോ കുറയ്ക്കാനോ ഉള്ള തന്ത്രങ്ങൾ വികസിപ്പിക്കുകയും നടപ്പിലാക്കുകയും ചെയ്യുക.

സ്ഥിരീകരണം: ദുർബലതകളെ പരിഹരിക്കുന്നതിന് പരിഹാര നടപടികൾ ഫലപ്രദമായിരുന്നുവെന്ന് ഉറപ്പാക്കുക.

തുടർച്ചയായ നിരീക്ഷണം: ദുർബലതകളുടെയും സാധ്യതയുള്ള ഭീഷണികളുടെയും നിരന്തരമായ ട്രാക്കിംഗ്, സ്ഥിരമായ വൾണറബിലിറ്റി സ്കാനുകളും മൂല്യനിർണ്ണയങ്ങളും നടത്തുക.

വൾണറബിലിറ്റി മാനേജ്മെന്റ് സൈബർസുരക്ഷയുടെ ഒരു നിർണായക ഘടകമാണ്, ഇത് സ്ഥാപനങ്ങൾക്ക് ശക്തമായ സുരക്ഷാ നിലനിൽപ്പ് നിലനിർത്താനും സുരക്ഷാ ലംഘനങ്ങളുടെ സാധ്യത കുറയ്ക്കാനും സഹായിക്കുന്നു. വൾണറബിലിറ്റി മാനേജ്മെന്റ് ടൂളുകളുടെ ഉദാഹരണങ്ങൾ: Defender for Cloud, Nessus, Qualys.

## CSPM എന്താണ്?

Cloud Security Posture Management (CSPM) എന്നത് ക്ലൗഡ് പരിസ്ഥിതികളിൽ സുരക്ഷാ ക്രമീകരണങ്ങളും മികച്ച രീതികളും വിലയിരുത്താനും നിരീക്ഷിക്കാനും നടപ്പിലാക്കാനും രൂപകൽപ്പന ചെയ്ത ഉപകരണങ്ങൾ, പ്രക്രിയകൾ, പ്രാക്ടീസുകൾ എന്നിവയുടെ ഒരു സമുച്ചയമാണ്. CSPM പരിഹാരങ്ങൾ, Amazon Web Services (AWS), Microsoft Azure, Google Cloud Platform (GCP) എന്നിവയുൾപ്പെടെയുള്ള ക്ലൗഡ് പ്രൊവൈഡർമാരുടെ ക്ലൗഡ് ഇൻഫ്രാസ്ട്രക്ചറിൽ ഉള്ള തെറ്റായ കോൺഫിഗറേഷനുകൾ, സുരക്ഷാ പാളിച്ചകൾ, അനുപാലന ലംഘനങ്ങൾ എന്നിവ തിരിച്ചറിയാനും പരിഹരിക്കാനും സ്ഥാപനങ്ങൾക്ക് സഹായിക്കുന്നു.

CSPM-ന്റെ പ്രധാന ഘടകങ്ങൾ:

- കോൺഫിഗറേഷൻ വിലയിരുത്തൽ: തെറ്റായ കോൺഫിഗറേഷനുകൾ, സുരക്ഷിതമല്ലാത്ത ക്രമീകരണങ്ങൾ, സുരക്ഷാ മികച്ച രീതികളിൽ നിന്ന് വ്യതിചലനം എന്നിവ തിരിച്ചറിയാൻ ക്ലൗഡ് റിസോഴ്സുകളും സേവനങ്ങളും സ്കാൻ ചെയ്യുക.

- സുരക്ഷാ നയം നടപ്പാക്കൽ: തെറ്റായ കോൺഫിഗറേഷനുകൾ സ്വയം പരിഹരിക്കുകയോ അലർട്ടുകൾ സൃഷ്ടിക്കുകയോ ചെയ്യുന്നതിലൂടെ ക്ലൗഡ് പരിസ്ഥിതികളിൽ സുരക്ഷാ നയങ്ങളും അനുപാലന മാനദണ്ഡങ്ങളും നടപ്പിലാക്കുക.

- തുടർച്ചയായ നിരീക്ഷണം: ക്ലൗഡ് ആസ്തികളുടെയും റിസോഴ്സുകളുടെയും സുരക്ഷാ നിലനിൽപ്പിൽ തുടർച്ചയായ ദൃശ്യപരത നൽകുക, സുരക്ഷാ സംഭവങ്ങൾക്ക് റിയൽ-ടൈം അലർട്ടുകൾ നൽകുക.

- അനുപാലന റിപ്പോർട്ടിംഗ്: നിയമപരമായ ആവശ്യകതകളും വ്യവസായ മാനദണ്ഡങ്ങളും അനുസരിക്കുന്നതിന്റെ തെളിവുകൾ നൽകുന്ന റിപ്പോർട്ടുകളും ഡാഷ്ബോർഡുകളും സൃഷ്ടിക്കുക.

CSPM സ്ഥാപനങ്ങൾക്ക് അവരുടെ ക്ലൗഡ് ഇൻഫ്രാസ്ട്രക്ചർ സുരക്ഷിതവും അനുപാലനക്ഷമവും മികച്ച രീതികളുമായി പൊരുത്തപ്പെടുന്നതും ഉറപ്പാക്കാൻ സഹായിക്കുന്നു, ക്ലൗഡുമായി ബന്ധപ്പെട്ട സുരക്ഷാ സംഭവങ്ങളുടെ അപകടസാധ്യത കുറയ്ക്കുന്നു. CSPM ടൂളുകളുടെ ഉദാഹരണങ്ങൾ: Defender for Cloud, Snyk.

## CNAPP എന്താണ്?

CNAPP എന്നത് Cloud-Native Application Protection Platform എന്നതിന്റെ ചുരുക്കമാണ്. ഇത് ക്ലൗഡ് സുരക്ഷാ ഭീഷണികളും ദുർബലതകളും നിരീക്ഷിക്കുകയും കണ്ടെത്തുകയും അവയിൽ പ്രവർത്തിക്കുകയും ചെയ്യുന്നത് ലളിതമാക്കുന്ന ഒരു ഓൾ-ഇൻ-വൺ ക്ലൗഡ്-നെറ്റീവ് സോഫ്റ്റ്വെയർ പ്ലാറ്റ്ഫോം ആണ്. ഇത് നിരവധി ഉപകരണങ്ങളും ശേഷികളും ഒരു സോഫ്റ്റ്വെയർ പരിഹാരത്തിൽ സംയോജിപ്പിച്ച് സങ്കീർണ്ണത കുറയ്ക്കുകയും DevOps, DevSecOps ടീമുകളുടെ പ്രവർത്തനങ്ങൾ എളുപ്പമാക്കുകയും ചെയ്യുന്നു. CNAPP CI/CD ആപ്ലിക്കേഷൻ ജീവിതചക്രത്തിന്റെ മുഴുവൻ ഘട്ടങ്ങളിലും, വികസനത്തിൽ നിന്ന് പ്രൊഡക്ഷൻ വരെ, സമഗ്രമായ ക്ലൗഡ്, ആപ്ലിക്കേഷൻ സുരക്ഷ നൽകുന്നു. ഇത് ആധുനിക ക്ലൗഡ് സുരക്ഷാ നിരീക്ഷണത്തിനും, പോസ്റ്റർ മാനേജ്മെന്റിനും, ലംഘന പ്രതിരോധത്തിനും നിയന്ത്രണ ഉപകരണങ്ങൾക്കും വ്യവസായത്തിന്റെ ആവശ്യകതയെ പരിഹരിക്കുന്നു. CNAPP പരിഹാരങ്ങൾ ക്ലൗഡ് പരിസ്ഥിതിയിലെ അപകടസാധ്യതകളെ അളക്കാനും പ്രതികരിക്കാനും സുരക്ഷാ ടീമുകൾക്ക് കഴിവ് നൽകുന്നു. CNAPP-ന്റെ ചില സവിശേഷതകൾ: മെച്ചപ്പെട്ട ദൃശ്യപരത, അപകടസാധ്യതകളുടെ അളവെടുപ്പ്, സുരക്ഷിത സോഫ്റ്റ്വെയർ വികസനം, സംയോജിത ക്ലൗഡ് സുരക്ഷാ പരിഹാരം. CNAPP-കൾ പരമ്പരാഗതമായി വേർതിരിച്ചിരുന്ന നിരവധി ക്ലൗഡ്-നെറ്റീവ് സുരക്ഷാ ഉപകരണങ്ങളും ഡാറ്റാ സ്രോതസ്സുകളും ഒരു ഉപയോക്തൃ ഇന്റർഫേസിൽ സംയോജിപ്പിക്കുന്നു, സ്ഥാപനങ്ങൾക്ക് അവരുടെ മുഴുവൻ ക്ലൗഡ് ആപ്ലിക്കേഷൻ ഫുട്പ്രിന്റ് സംരക്ഷിക്കാൻ എളുപ്പമാക്കുന്നു. CNAPP-കൾ കണ്ടെയ്നർ സ്കാനിംഗ്, ക്ലൗഡ് സുരക്ഷാ പോസ്റ്റർ മാനേജ്മെന്റ്, ഇൻഫ്രാസ്ട്രക്ചർ എസ് കോഡ് സ്കാനിംഗ്, ക്ലൗഡ് ഇൻഫ്രാസ്ട്രക്ചർ എന്റൈറ്റിൽമെന്റ്സ് മാനേജ്മെന്റ്, റൺടൈം ക്ലൗഡ് വർക്ക്‌ലോഡ് പ്രൊട്ടക്ഷൻ പ്ലാറ്റ്ഫോം എന്നിവ ഉൾപ്പെടെ നിരവധി ക്ലൗഡ്-നെറ്റീവ് സുരക്ഷാ ഉപകരണങ്ങളും ഡാറ്റാ സ്രോതസ്സുകളും സംയോജിപ്പിക്കുന്നു.
